# Terminal commands

Here are some terminal commands we use

## Installation
```bash
# klon repo
git clone git@github.com:user/repo.git
git clone github.com/user/repo.git

# push applikation til github
git init
git add .
git commit -m "commit besked"
git push origin master

# installer dependencies
npm install
npm i

# installer dependencies globalt
npm install -g
npm i -g

# kør dev
npm run dev

# kør build
npm run build

# opret package.json
npm init

# installer Express for at oprette package-lock.json
npm install express
npm i express

# auto genstart applikation når ændringer opfanges
nodemon app.js
